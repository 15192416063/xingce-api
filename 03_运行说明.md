# 最小可用题库系统 app.py — 运行说明

## 这个功能是什么
一个网页应用，实现你要的核心功能：
- **管理员（你）上传 PDF → 题目进入公共题库**
- **所有用户打开网页 → 就能搜公共题库里的相似题**
- 按「考点语义」匹配，不是关键词匹配

## 怎么运行
1. 装依赖：
   pip install streamlit pdfplumber langchain langchain-community langchain-openai chromadb
2. 设置模型 key（用OpenAI时）：
   export OPENAI_API_KEY="你的key"   （Windows: $env:OPENAI_API_KEY="你的key"）
3. 启动：
   streamlit run app.py
   → 浏览器自动打开，本地即可体验

## 怎么用
- 「🔑 管理员上传」标签页：输入口令(默认admin123)，上传PDF，题目存入公共库
- 「🔍 找相似题」标签页：任何人输入一道题，返回考点最相似的题

## 重要架构说明（给技术人员）
- 每条题带 scope 标记：public=公共库(所有人可搜)，预留 user:xxx=某用户私有库
- 检索时按 scope 过滤。当前只搜 public。后续加私有题库：
  入库时 scope 设成 user:用户id，检索时 filter 改成 {"scope": {"$in": ["public", "user:当前用户id"]}}
- 这样「公共开箱库 + 用户私有库」并存，就是成熟产品的形态

## 上线前必改
1. ADMIN_PASSWORD（代码顶部）改掉，换成正规登录鉴权
2. 本地 Chroma 换成 Qdrant/Milvus 等支持多用户的向量库（见技术方案书第八节）
3. 模型可换成国产（通义/智谱/DeepSeek），改 get_models() 即可

## 关于题库内容（再次提醒）
这个程序只是「能上传题、能检索」的功能。
往公共库里放什么题，是内容来源问题：
请务必使用 ① 你自己原创/AI生成 ② 采购授权 的干净题目，
不要放网上抓来的机构整理真题，避免版权风险。
