# 行测相似题智能检索系统 — 项目交付包

本包是「行测相似题检索系统」的完整交付物，供开发团队参考实现。

## 核心功能
用户给出一道行测题（或主题），系统按「考点的语义相似度」从题库中找出最相似的题。
区别于关键词搜索：字面不同但考点相同的题也能匹配到。
题库分两种：公共题库（管理员上传，全员可搜）+ 用户私有题库（仅本人可搜）。

## 快速上手路径

- **只想看效果** → 按 03_运行说明.md 跑起 app主程序.py，体验上传+检索全流程。
- **交给技术人员** → 把 01_开发提示词.md + 02_技术方案书.md + app主程序.py 三样一起给他，足够动手。

## 三个最重要的提醒

1. **核心设计**：用 LLM 提炼的「考点摘要」算向量，不是用原题算。这决定检索效果好坏，详见 01_开发提示词.md 第〇节。
2. **双库隔离**：每条题带 scope 标记（public / user:xxx），检索按 scope 过滤。私有题隔离是隐私红线。
3. **版权红线**：公共题库内容必须来源干净（原创/AI生成/正版采购），不可放网上抓的机构整理真题。上线前请咨询律师。
