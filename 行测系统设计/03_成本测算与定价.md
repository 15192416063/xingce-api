# 成本测算与定价建议

> ⚠️ 单价随服务商调整频繁,下面是**2026年初的代表性价格,上线前务必去各官网重核**。
> 这份文档的价值不在精确数字,而在**成本结构和定价逻辑**——结论很稳:AI 成本极低,不是瓶颈。

| 服务 | 用途 | 代表单价 |
|---|---|---|
| DeepSeek `deepseek-chat` | 切题 / 分类 / 考点提炼 | 输入 ≈ ¥2 / 百万token,输出 ≈ ¥8 / 百万token |

> 经验法则:1 个汉字 ≈ 0.6–1 token,这里按 1 字≈1 token 粗估(偏保守,真实更低)。

## 二、单题入库成本(一次性)

### 1) 普通文字题(言语/常识/逻辑/类比/定义/数量)
| 步骤 | token 估算 | 成本 |
|---|---|---|
| 切题(分批摊销,每批~3000字出5–8题) | 摊到每题 输入~500 / 输出~200 | ≈ ¥0.0026 |
| 分类+考点提炼 | 输入~400 / 输出~120 | ≈ ¥0.0018 |
| 算向量(考点摘要~50字) | 50 token | ≈ ¥0.00003 |
| **合计** | | **≈ ¥0.004 / 题** |

> 即:**约 250 道文字题 ≈ 1 元**。

### 2) 资料分析题(材料+小题+图表)
- 文字部分同上 ≈ ¥0.004 / 小题(材料背景让输入略大,算 ¥0.006)。
- 图表多模态:**每个材料组只识别一次**(一组图共用),约 ¥0.015 / 组,摊到 4–6 道小题 ≈ ¥0.003 / 题。
- **合计 ≈ ¥0.01 / 题**。

### 3) 图推题(题干是图)
| 步骤 | 成本 |
|---|---|
| qwen-vl 整页识规律(单题或整页摊) | ≈ ¥0.04–0.08 / 题 |
| 算向量(规律文字) | ≈ ¥0.00003 |
| **合计** | **≈ ¥0.05–0.08 / 题** |

> 图推是文字题的 **15–20 倍贵**(demo 文档说 30–50 倍是更保守的上限)。**这是唯一要用额度卡死的成本项。**

### 4) 检索 / 出题成本
- "按题型出题" = 纯 MySQL 查询,**¥0 AI 成本**。
- "找相似题" = 1 次 embedding ≈ ¥0.00003,可忽略。
- **结论:用户日常做题几乎零边际成本。** 成本全压在"入库一次"。

## 三、典型场景测算

### 场景A:用户传 1 套国考真题(约 130 题,如样本卷)
约 110 文字 + 13 资料分析 + 7 图推:
```
110 × 0.004  +  13 × 0.01  +  7 × 0.06
= 0.44 + 0.13 + 0.42 ≈ ¥0.99
```
**一套完整真题入库 ≈ 1 元(一次性)。**

### 场景B:重度用户传 2000 题(含 200 图推)
```
1800 × 0.004  +  200 × 0.06
= 7.2 + 12 ≈ ¥19.2(一次性)
```
之后该用户每月做题成本 ≈ **¥0**。

## 四、成本结构结论

| 成本项 | 量级 | 是否瓶颈 |
|---|---|---|
| AI 推理(入库) | 几毛~几十元/重度用户,一次性 | ❌ 不是瓶颈 |
| AI 推理(检索) | 趋近于 0 | ❌ |
| **图推多模态** | 单题贵 15–20 倍 | ⚠️ 唯一需额度管控 |
| 服务器(云主机) | 固定月费 | ✅ 真正的固定成本 |
| MinIO 对象存储 | 随 PDF/图片量增长 | ✅ 需关注 |
| 支付通道手续费 | 约 0.6% / 笔 | 小 |
| ICP备案/域名/SSL | 一次性+年费 | 小 |

**一句话:AI 不是成本中心,服务器+存储才是固定支出。毛利空间极大。**

## 五、定价建议

定价**挂在"上传额度 + 图推额度",不是单纯按时间**——因为成本来自上传量。

| 档位 | 价格 | 上传额度 | 图推额度 | 其他 |
|---|---|---|---|---|
| 免费版 | ¥0 | 200 题(约1.5套卷) | 5 题 | 基础出题+错题本,引流验证 |
| 月度会员 | ¥19 / 月 | 3000 题 | 100 题 | AI解析、数据统计、相似强化 |
| 年度会员 | ¥149 / 年 | 30000 题 | 1000 题 | 全功能 + 智能组卷 |
| 超额加油包 | ¥9 / 1000题 | 叠加 | — | 重度用户增收 |

### 毛利估算(月度会员 ¥19)
- 一个月度用户即使用满 3000 题(含 100 图推)入库:
  `2900×0.004 + 100×0.06 ≈ 11.6 + 6 ≈ ¥17.6` 一次性 AI 成本。
- **但绝大多数用户用不满额度**,真实 AI 成本通常 < ¥3/人。会员费扣掉支付手续费后,毛利 80%+。
- 风险点:**有人薅满图推额度**——所以图推额度必须卡死(100题/月),且单独计费加油包。

## 六、控成本的三条铁律

1. **贵操作只在入库时做一次,结果落库**;用户检索/做题永远不实时调贵模型。
2. **图推/资料分析多模态用额度硬卡**,免费版给极少量(5题)防薅。
3. **加缓存**:相同题指纹去重(不重复入库扣费);常见查询结果走 Redis 缓存。

> 最后提醒:真正威胁盈利的不是 token 成本,是**没人付费**。先用免费版把"找相似题/按类型练"的价值跑出来,再谈优化成本。
